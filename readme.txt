Auto Select Theme With Conection Speed. For WP

== Description ==
Auto Select Theme With Conection Speed.
Select Theme for DialUP! & DSL cat, go to 'Setting' Menu->Barnameha CSTS

== Installation ==
1. Download and extract the `barnameha-csts` plugin file
1. Upload `barnameha-csts` directory  to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Plugin Config: 'Setting' Menu -> Barnameha CSTS

== Frequently Asked Questions ==

= How Work? =
In the first call, your user speed is stored.
Then the next call immediately stored template is selected.
